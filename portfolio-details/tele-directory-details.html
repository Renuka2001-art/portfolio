Tele Directory App

The Tele Directory is a lightweight internal web application designed for HR and department leads to manage employee contact records seamlessly. With a modern interface, fast search capability, and smooth CRUD operations, this tool increases efficiency across the organization.

  • ✅ Built using HTML5, CSS3, Bootstrap 5, and Vanilla JavaScript
  • ✅ Add, edit, and delete contact entries with zero page reload
  • ✅ Live search to filter contacts in real-time
  • ✅ Validates user input before adding to the directory
  • ✅ Responsive layout optimized for desktop and mobile views
  • ✅ Clean and intuitive design for non-technical users
