Tele Directory App

Tele Directory is an internal HR tool that simplifies contact management. It allows HR and team leads to quickly access, add, or remove employee contact information without reloading the page.

  • ✅ Built with HTML, CSS, Bootstrap, JavaScript
  • ✅ Add/delete contacts dynamically
  • ✅ Fully responsive design
  • ✅ Smooth user experience for HR use
